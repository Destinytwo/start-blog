# 关于我

你好，我是 **Funny**。

我关注的方向是计算机视觉、无人机目标检测和工程化系统开发。这个网站主要用来整理我的项目、论文笔记和技术复盘，也作为一个持续更新的个人技术档案。

## 我的方向

我目前主要围绕两条主线积累：

- **无人机小目标检测**：关注航拍图像中的小目标、弱纹理目标和密集目标检测问题，整理过 UAV-SmallDet、RT-DETR、HG-PKINet、STFL-Neck、SA-NWD 等相关方法和实验经验。
- **建筑缺陷检测系统**：面向建筑外立面巡检场景，把无人机航拍影像、实景三维模型、目标检测、人工复核和报告导出串成完整流程。

我更关心算法如何进入真实系统，而不只是停留在单次实验结果上。对我来说，一个完整项目通常需要同时处理数据、模型、界面、三维可视化、部署和交付文档。

## 我做过什么

这个网站里记录的项目和文章，大多来自以下实践：

- 使用 SAHI、YOLO/Ultralytics 等方案处理高分辨率建筑外立面缺陷检测。
- 基于 PyQt 和 FastAPI 搭建检测客户端与推理服务分离的桌面系统。
- 使用 OSGB、OpenSceneGraph、空三参数实现二维照片和三维模型联动。
- 将二维检测框映射到三维模型中的缺陷点，并处理多照片重复缺陷合并。
- 集成 LabelMe 做人工补标和复核，让模型检测结果可以被修正、追踪和写回系统。
- 生成楼层切面图、缺陷统计表和 Word 检测报告，把检测结果变成可交付材料。
- 整理 UAV-SmallDet 相关论文写作、实验表格、LaTeX 排版和可视化复盘。

## 我的技术栈

我常用的技术包括：

- Python、PyQt、FastAPI、SQLite
- Ultralytics、SAHI、RT-DETR、小目标检测相关模型
- LabelMe、数据集清洗、标注转换和标注质检
- OSGB、OpenSceneGraph、实景三维模型加载与坐标映射
- LaTeX、技术文档、报告生成和项目复盘

## 这个网站记录什么

这里不是单纯的作品展示页。我更希望它像一个可追溯的工作记录：把一个系统从想法、数据准备、算法验证、界面设计、工程踩坑到报告输出的过程拆开写清楚。

如果你想快速了解我，可以从这些主题开始：

- UAV-SmallDet：无人机小目标感知网络
- 基于实景三维模型的建筑物缺陷检测系统
- SAHI 在建筑外立面缺陷检测中的作用
- 二维照片和三维模型联动
- PyQt + FastAPI 检测系统架构

我会继续把项目中遇到的具体问题写下来，包括模型为什么这样设计、系统为什么这样拆分、某个工程问题最后是怎么解决的。
